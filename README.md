# Smile Dental Clinic Website

A modern, professional dental clinic homepage built with React.js and Tailwind CSS.

## Features

- **Clean & Professional Design** - Premium medical theme with Medical Blue (#1E88E5) and Teal Accent (#00BFA5)
- **Fully Responsive** - Mobile-first design that works on all devices
- **Modern UI** - Soft shadows, rounded corners, smooth hover transitions
- **Component-Based** - Reusable React components for maintainability

## Sections

1. **Hero** - Full-width headline with CTAs and doctor image
2. **About** - Clinic experience, stats (10+ years, 5000+ patients, 100% sterilized)
3. **Services** - 6 service cards (Cleaning, Root Canal, Implants, Braces, Cosmetic, Pediatric)
4. **Meet the Dentist** - Doctor profile and qualifications
5. **Why Choose Us** - 6 feature highlights
6. **Before & After Gallery** - Image comparison cards
7. **Testimonials** - 3 patient reviews
8. **Appointment Booking** - Contact form
9. **Contact & Location** - Address, phone, hours, map placeholder
10. **Footer** - Quick links, services, social media

## Setup

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- Poppins & Inter fonts (Google Fonts)
